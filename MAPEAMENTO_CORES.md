# Mapeamento de Variáveis CSS - Cores.css

## Substituições Necessárias

### Cores Base
- `--white` → `--color-white`
- `--black` → `--color-black`
- `--highlight` → `--color-highlight`

### Backgrounds
- `--bcsite` → `--color-bg-site`
- `--black12` → `--color-bg-dark`
- `--black1d` → `--color-bg-darker`
- `--gray1e` → `--color-bg-section`
- `--focus` → `--color-focus`

### Cinzas
- `--gray3c` → `--color-gray-medium`
- `--textfade` → `--color-gray-light`
- `--d1d1d1` → `--color-gray-lighter`

### Bordas
- `--borders` → `--color-border`
- `--bordersopacity` → `--color-border-light`

### Opacidades
- `--highlightopacity` → `--color-highlight-opacity`
- `--blackopacity` → `--color-black-opacity`

### Gradientes
- `--section` → `--bg-section`
- `--bcplusradial` → `--bg-card`
- `--bcradial` → `--bg-radial`
- `--bcradialhover` → `--bg-radial-hover`
- `--bcblackradial` → `--bg-radial-dark`
- `--bcicon` → `--bg-icon`
- `--bcbtn` → `--bg-button`
